## Token Rate Limit Filter
A simple yet powerful function that lets admins enforce configurable token limits per user within a set time window. Admins are exempt by default.

GitHub: https://github.com/ArtemYatsyuk/Token-Rate-Limit-Filter

Additional add-ons are in development. Follow the repository on GitHub and leave a ⭐ to support the project.

Please note: this function will not receive further updates — new features will ship as standalone add-ons.
